In this Project we are tasked with creating a functional roster of students for the University.

For this project we have created 4 objects for data, and 1 custom data structure as our functioning roster.

For this project we utilized the Java Programming language and its capabilites of Object-Oreiented Programming (OOP).

This program is a user interactive program through the command line. The basic structure of inputs to the program consist of an Operation code, followed by the required data for that specific operation.

The set of basic commands for the program are as follows:

A -> add student
Ex: A John Doe 1/20/2003 CS 101
R -> remove student
Ex: R John Doe 1/20/2003
P -> print sorted roster by name, DoB
PS -> print sorted roster by standing
PC -> print sorted roster by school, major
L -> print list of students in a school
Ex: L SAS
C -> change student major
Ex: C John Doe 1/20/2003 CS
Q -> terminate program
